  • Architecture

      • The Azure Architecture Center contains guidance for building end-to-end solutions on Microsoft Azure. Here you will find reference architectures, best practices, design patterns, scenario guides, and reference implementations.

      • Azure Architecture Center

        Guidance for building end-to-end solutions on Microsoft Azure.

      • Azure Application Architecture Guide

        This guide presents a structured approach for designing applications on Azure that are scalable, resilient, and highly available. It is based on proven practices that we have learned from customer engagements.

      • Reference Architectures

        A set of recommended architectures for the cloud. Each architecture includes best practices, prescriptive steps, and a deployable solution.

      • Cloud Design Patterns

        Design patterns for the cloud. Each pattern describes a problem, a pattern that addresses the problem, and an example based on Azure.
